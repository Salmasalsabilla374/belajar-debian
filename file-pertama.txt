**Dasar dasar linux**
Dasar-dasar Linux mencakup pemahaman tentang perintah-perintah dasar yang digunakan untuk mengelola sistem operasi Linux. Beberapa perintah dasar yang penting antara lain ls (melihat isi direktori), cd (berpindah direktori), pwd (menampilkan direktori saat ini), mkdir (membuat direktori), rmdir (menghapus direktori kosong), rm (menghapus file atau direktori), cp (menyalin file atau direktori), mv (memindahkan atau mengganti nama file/direktori), dan touch (membuat file kosong). 
Berikut adalah beberapa perintah dasar Linux yang umum digunakan: 

    ls:
    Perintah ini digunakan untuk menampilkan daftar file dan direktori di direktori saat ini. Contohnya, ls akan menampilkan semua file dan folder yang ada di direktori tempat Anda berada.
    cd:
    Perintah ini digunakan untuk berpindah dari satu direktori ke direktori lainnya. Misalnya, cd Documents akan membawa Anda ke direktori "Documents".
    pwd:
    Perintah ini menampilkan direktori kerja saat ini. Ini membantu Anda mengetahui lokasi Anda di struktur direktori sistem.
    mkdir:
    Perintah ini digunakan untuk membuat direktori baru. Misalnya, mkdir contoh_direktori akan membuat direktori baru bernama "contoh_direktori".
    rmdir:
    Perintah ini digunakan untuk menghapus direktori kosong. Perintah ini hanya bisa menghapus direktori yang kosong, tidak berisi file atau subdirektori.
    rm:
    Perintah ini digunakan untuk menghapus file atau direktori. Berhati-hatilah saat menggunakan perintah ini, karena file atau direktori yang dihapus tidak dapat dikembalikan.
    cp:
    Perintah ini digunakan untuk menyalin file atau direktori. Contohnya, cp file.txt /home/user/dokumen/ akan menyalin file "file.txt" ke direktori "/home/user/dokumen/".
    mv:
    Perintah ini digunakan untuk memindahkan atau mengganti nama file atau direktori. Misalnya, mv file.txt file_baru.txt akan mengganti nama "file.txt" menjadi "file_baru.txt".
    touch:
    Perintah ini digunakan untuk membuat file kosong baru. Jika file sudah ada, perintah ini hanya akan memperbarui stempel waktu file tersebut.
    cat:
    Perintah ini digunakan untuk menampilkan isi file di terminal. Contohnya, cat nama_file.txt akan menampilkan isi file "nama_file.txt".
    grep:
    Perintah ini digunakan untuk mencari teks tertentu dalam file. Misalnya, grep "keyword" nama_file.txt akan mencari baris dalam "nama_file.txt" yang mengandung kata "keyword".
    man:
    Perintah ini digunakan untuk menampilkan manual (help) dari perintah lain. Contohnya, man ls akan menampilkan manual untuk perintah ls.
    sudo:
    Perintah ini digunakan untuk menjalankan perintah dengan hak akses superuser (root). Perintah ini membutuhkan kata sandi pengguna.
    nano
    atau vi: Editor teks yang umum digunakan di Linux. Nano lebih mudah digunakan untuk pemula. 

Selain perintah-perintah dasar di atas, ada banyak perintah lain yang berguna dalam Linux untuk berbagai keperluan, seperti manajemen file, jaringan, sistem, dan lain-lain. Memahami perintah-perintah dasar ini adalah langkah awal yang penting untuk menguasai Linux.

**Manajemen paket linux**
Proses instalasi, pembaruan, konfigurasi, dan penghapusan paket perangkat lunak dengan manajer paket di OS Linux disebut manajemen paket. Untuk menjaga sistem yang andal dan konsisten, tugas-tugas ini dapat diotomatisasi dan dependensi dapat dikelola. Manajer paket menyediakan akses ke repositori perangkat lunak terpusat yang menawarkan beragam pilihan program siap pakai. Instalasinya mudah. Versi perangkat lunak dapat diperiksa, dan paket dapat diperbarui secara otomatis. Instalasi dependensi yang diperlukan juga otomatis. Manajer paket dapat mengidentifikasi dan menyelesaikan konflik antar paket. Beberapa manajer paket dapat melakukan rollback ke versi paket sebelumnya atau membuat cadangan status paket. Beberapa manajer paket yang paling banyak digunakan di Linux adalah sebagai berikut:

    APT (Advanced Package Tool) : Digunakan oleh distribusi berbasis Debian seperti Ubuntu dan Mint.

    YUM (Yellowdog Updater, Modified) : Digunakan oleh distribusi berbasis Red Hat seperti RHEL dan CentOS.

    DNF (Dandified YUM) : Manajer paket generasi berikutnya untuk distribusi berbasis Red Hat, menggantikan YUM.

    Pacman : Digunakan oleh Arch Linux dan turunannya.

    Zypper : Digunakan oleh distribusi berbasis SUSE seperti openSUSE.

    Portage : Digunakan oleh Gentoo Linux dan turunannya

karena kita pakai debian 13 jadi jika fokus saja pada paket manajemen debian:

**konfigurasi sistem debian 13**
Debian 13, yang dikenal dengan nama kode "Trixie", adalah rilis terbaru dari sistem operasi Debian yang dirilis pada 9 Agustus 2025. Sistem ini menawarkan stabilitas dan dukungan perangkat keras yang luas, menjadikannya pilihan populer untuk server dan desktop. Debian 13 memiliki siklus hidup lima tahun, dengan tiga tahun dukungan penuh dan dua tahun Dukungan Jangka Panjang (LTS). 
Konfigurasi Sistem Debian 13
Berikut adalah beberapa aspek konfigurasi sistem yang umum pada Debian:

    1. Konfigurasi Jaringan:
        Debian menggunakan NetworkManager untuk mengelola koneksi jaringan. 

Anda dapat mengonfigurasi jaringan melalui antarmuka grafis atau baris perintah menggunakan nmcli. 
Untuk pengaturan DNS, NetworkManager biasanya mengelola /etc/resolv.conf. 
Pilih "Alamat Otomatis (DHCP) saja" untuk pengaturan jaringan otomatis, dan masukkan server DNS di kolom yang sesuai. 

2. Manajemen Paket:

    Debian menggunakan sistem manajemen paket APT (Advanced Package Tool). 

Anda dapat menginstal, memperbarui, dan menghapus perangkat lunak menggunakan perintah seperti apt update, apt upgrade, dan apt install <nama_paket>. 

3. Konfigurasi Sistem Dasar:

    Debian memungkinkan konfigurasi berbagai aspek sistem, seperti bahasa, zona waktu, dan keyboard melalui alat dpkg-reconfigure. 

Anda juga dapat mengonfigurasi sistem secara manual dengan mengedit file konfigurasi yang sesuai di direktori /etc. 

4. Keamanan:

    Debian dikenal dengan pembaruan keamanan berkala. 

Pastikan untuk selalu memperbarui sistem Anda untuk mendapatkan patch keamanan terbaru. 
Pertimbangkan untuk mengaktifkan firewall seperti ufw untuk melindungi sistem Anda dari akses yang tidak sah. 

5. Dukungan Perangkat Keras:

    Debian mendukung berbagai arsitektur perangkat keras, termasuk 64-bit dan 32-bit. 

Debian juga mendukung berbagai perangkat keras secara luas. 

6. Instalasi Debian:

    Anda dapat menginstal Debian dari gambar ISO yang dapat diunduh dari situs web resmi Debian. 

Debian dapat diinstal pada berbagai jenis perangkat keras, termasuk virtual machine. 

Informasi Tambahan:

    Debian 13 "Trixie" memiliki siklus hidup lima tahun, dengan dukungan penuh selama tiga tahun dan Dukungan Jangka Panjang (LTS) selama dua tahun. 

Debian dikenal dengan stabilitasnya, menjadikannya pilihan yang baik untuk server dan desktop yang membutuhkan keandalan. 
Debian juga menawarkan berbagai macam perangkat lunak yang dapat diinstal melalui sistem manajemen paket APT.
